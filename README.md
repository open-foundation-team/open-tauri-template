## Requirements

The following is a list of minimum requirements to run this project.

- [Rust](https://www.rust-lang.org/tools/install)
- [Node 24.12 [Current LTS]](https://nodejs.org/en/download/)

I recommend installing Rust using the [official installer](https://www.rust-lang.org/tools/install) or via brew and managing your Node.JS version using [nvm](https://github.com/nvm-sh/nvm).

## Getting started

1. Clone the repository
2. Install dependencies `npm install`
3. Run the project 
   1. Desktop `npm run dev:desktop`
   2. Web `npm run dev`
